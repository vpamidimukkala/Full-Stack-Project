**Football Club Management System (Full-Stack CRUD App)**

A full-stack web application to manage football clubs, allowing users to create, view, update, and delete club records through a responsive and intuitive user interface. 
The project demonstrates data storytelling, front-end development with React, and backend API design with Django REST Framework.


**Features**

Full CRUD operations for football club records (Create, Read, Update, Delete)

Modular and reusable React components with form validation

RESTful APIs using Django REST Framework (DRF) for backend operations

Integration between frontend and backend using HTTP methods

Structured error handling and API response validation for reliability

Scalable and maintainable architecture following MVC principles and REST standards



**Tech Stack**

**Frontend:**

React.js, JavaScript, HTML, CSS, Axios / Fetch API for HTTP requests

**Backend:**

Django, Django REST Framework (DRF)

Django CORS Headers

**Database:**

SQLite (default)


**Testing**

Tested all CRUD operations for correct functionality

API response validation and error handling verified

Edge cases and negative scenarios tested

Frontend components and forms validated for proper state handling
